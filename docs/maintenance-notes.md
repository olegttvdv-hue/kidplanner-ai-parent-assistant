# Maintenance Notes

This file keeps lightweight maintenance rules for KidPlanner AI Parent Assistant.

## Regular Checks

- Keep Telegram and payment examples as demo values.
- Review daily-limit behavior after workflow changes.
- Check subscription and request-count logic after database changes.
- Keep parent-facing copy clear and safe.
- Update product notes when scope changes.

## Safe Data Rules

Use placeholders for Telegram tokens, chat IDs, user IDs, payment provider data, webhook URLs, database passwords, and private parent messages.

## Release Hygiene

Before publishing a new update, check README accuracy, product overview, setup instructions, screenshots, privacy notes, and payment disclaimers.
